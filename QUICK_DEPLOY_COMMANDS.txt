STEP 2: DEPLOY TO RENDER (Make it Public)
------------------------------------------

1. Go to: https://render.com
2. Sign up with GitHub
3. Click "New +" → "Web Service"
4. Connect repository: prihhan/fuelprice-tallinn
5. Configure:

   Name: fuelprice-tallinn
   Runtime: Node
   Build Command:
   npm install && cd client && npm install && cd .. && npm run db:push && npm run db:generate && cd client && npm run build

   Start Command:
   npm start

   Environment Variables:
   DATABASE_URL=file:./dev.db
   NODE_ENV=production
   PORT=3001
   SCRAPER_INTERVAL_MINUTES=30

6. Click "Create Web Service"
7. Wait 5-10 minutes
8. Your app will be live at: https://fuelprice-tallinn.onrender.com

TROUBLESHOOTING
---------------

If git push fails:
- Use Personal Access Token instead of password
- Get token from: https://github.com/settings/tokens

If deployment fails:
- Check Render logs
- Verify environment variables
- Check build command syntax

If app doesn't load:
- Wait 10 minutes for first deployment
- Check Render dashboard for status
- Test API: https://your-app.onrender.com/api/health
